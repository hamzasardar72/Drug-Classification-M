# Drug Classification Using Machine Learning

## Overview
Built and compared 5 ML models to predict appropriate 
drug for patients based on medical attributes.

## Models Used
- Logistic Regression
- Decision Tree
- Random Forest ✅ (Best Model)
- KNN
- SVM

## Dataset
Drug200 Dataset — 200 patient records
Features: Age, Sex, BP, Cholesterol, Na_to_K

## Technologies
Python | Scikit-learn | Pandas | Matplotlib | Seaborn

## Tasks Covered
1. EDA
2. Data Preprocessing
3. Train-Test Split
4. Model Training
5. Model Evaluation
6. Model Comparison
7. Prediction Testing
8. Bonus: ML Pipeline# Drug-Classification-M
Drug Classification using ML models - LR, DT, RF, KNN, SVM
